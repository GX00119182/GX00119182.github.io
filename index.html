Projects

This app is being built as an Android application written in Java. The idea is for public transport inspectors to be able to detect when someone has not paid thier fare. The app itself will record offenders details, take a front-facing picture of the offender and check for a match in a backend database. It will use facial recognition to determine if the user is in the database or not or if they are in it but not with the same name or address. This will help the company to target offenders better. It will calculate the offenders fine based on the number of offences they have made prior to this one. It will also draw up analysis based on the data collected using R programming.

The system designed is for a local council or private library. The idea was to come up with a more user friendly way of dealing with day to day transactions of a library through self-service library loan system. The idea would be that the user would log on using thier username and password, and get access to books they wish to rent. Books would be loaned out and using an algorithm created by my team and I, would calculate the loan period based on the books popularity. It would also issue fines for late returns and allow for credit card payments online thus reducing the need f.or humsn interaction with a librarian.

The Tableau project was baes around the idea of visualising data in an informative way so that any user can read it and understand ehat the data is trying to tell. The task was to filter, mine and clean an excel spreadsheet of data and create calculated fields, etc, for visualisation and show these different visualisations using a dashboard.
